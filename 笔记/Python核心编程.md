  1. print 语句及“Hello World!”

通常当你想看变量内容时，你会在代码中使用 print 语句输出。不过在交互式解释器中，
你可以用 print 语句显示变量的字符串表示，或者仅使用变量名查看该变量的原始值。
在下面的例子中，我们把一个字符串赋值给变量 myString，先用 print 来显示变量的内容，
之后用变量名称来显示。

``` python
>>> myString = 'Hello World!'
>>> print myString
Hello World!
>>> myString
'Hello World!'
```
